Overview

This repository contains Python programs developed as part of academic assignments. The programs cover a wide range of topics including arithmetic operations, text processing, formatted output, and file handling. Each program demonstrates clear logic, proper variable naming, and handles edge cases effectively.

Design Principles:
                  Programs are structured for readability, maintainability, and step-by-step logic.
                  Edge cases and invalid inputs are handled gracefully using try-except.
                  Consistent formatting and variable naming enhance usability and learning clarity.

                  
Learning and Challenges:
                      Challenges included aligning formatted outputs and validating user inputs.
                      The programs demonstrate practical Python concepts such as loops, conditions, functions, string handling, and file I/O.
                      Testing with multiple inputs ensured robustness and correctness.
